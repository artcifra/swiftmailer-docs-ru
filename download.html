Установка

The preferred way to install Swiftmailer is via Composer: 

$ composer require swiftmailer/swiftmailer

Source Code Repository Access

If you want to keep up-to-date with the very latest developments as they happen you can install and use Swift Mailer directly from our source code repository, which currently uses Git and is hosted over at GitHub.
